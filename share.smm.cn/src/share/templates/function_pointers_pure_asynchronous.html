{{body_title("函数指针的基本概念")}}

函数指针(指向函数的指针／存储函数地址的变量)

函数是基本数据类型

函数像 array ， struct ， object 等复合数据类型一样，可以被当作数据一样被创建、赋值、传递。

javascript : 


        function a (){
            console.log("hello world!");
        }

        var b = a;

        b();

go : 


        func a (){
            log.Println("hello world!")
        }

        b := a

        b()

几乎所有的语言都支持

许多语言都直接支持函数指针，如 c ， go ， javascript 。

哪些没有直接支持函数指针的语言，也会通过类似于接口，面向对象等方式间接提供（封装）函数指针功能，如 java 。
