函数指针

定义

函数指针指一个指向函数的指针，即一个存储着函数地址的变量。

引申定义

当一个语言支持函数指针时，就意味着函数已经不仅仅是个语法层面的概念，而是一个数据层面的概念。

函数像 array ， struct ， object 等复合数据类型一样，可以被当作数据一样被创建、赋值、传递、使用。

简单示例


        func a (){
            log.Println("hello function!")
        }

        func main(){

            b := a

            b()
        }

语言支持

许多语言都直接支持函数指针，如 c ， go ， javascript 。

哪些没有直接支持函数指针的语言，也会通过类似于面向对象等方式间接提供（封装）函数指针功能，如 java 。
