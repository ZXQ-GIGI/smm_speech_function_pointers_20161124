回调函数(传递给他人，供他人在合适的时机调用的函数)

定义

以指针的方式传递给另一个函数，供对方在合适的时机调用的函数，称为回调函数。

简单示例


        func a (text string){

            log.Println(text)

        }

        func b (callback func(string)){

            callback("hello function!")

        }

        func main(){

            b(a)

        }

匿名函数作为回调函数


        func b (callback func(string)){

            callback("hello function!")

        }

        func main(){

            b(func (text string){

                log.Println(text)

            })
        }

回调函数的回调函数


        func main(){

            b(func (func () ){

                log.Println(text)

            },true)
        }
