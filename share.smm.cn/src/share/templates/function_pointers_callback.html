回调函数

定义

以指针的方式传递给另一个函数，供对方在合适的时机调用的函数，称为回调函数。

简单示例


        func a (text string){

            fmt.Println(text)

        }

        func b (callback func(string)){

            callback("hello function!")

        }

        func main(){

            b(a)

        }

匿名函数作为回调函数


        func b (callback func(string)){

            callback("hello function!")

        }

        func main(){

            b(func (text string){

                fmt.Println(text)

            })
        }

回调函数的回调函数


        func b ( callback func( func () string ) ){

            callback(func()string{

                return "hello function!"

            })
        }

        func main(){

            b(func( text func () string ){

                fmt.Println( text() )

            })
        }
